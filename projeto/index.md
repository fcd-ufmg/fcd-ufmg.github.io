## Sobre

Um dos pontos essencias do curso de FCD é aplicar o conhecimento de sala de aula em uma base de dados
de interesse público. Portanto, neste projeto vamos usar as técnicas de ciência de dados para gerar
um relatório em notebook Jupyter.  O importante do projeto é que o mesmo analise uma base de dados seguindo todo o
ciclo do cientista de dados. Ou seja, é uma análise além do trivial (computar uma média)
